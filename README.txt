## Introduction
CCHabitPro helps you track habits for chronic diseases like hypertension, obesity, and diabetes. The app allows users to store personal information and habits aimed at improving their health.

## Installation

**Requirements:** 
Make sure you have Python 3.12 installed on your computer. You can download the latest version of Python [here](https://www.python.org/downloads/). 

**Req. Package:**
* [questionary](https://github.com/tmbo/questionary) (install via "pip install questionary")

**How To:**
1. Download the files `Habit.py`, `initialisation.py`, `main.py`, and `User.py`.
2. Save them in a folder on your computer.
3. Open your terminal and run the command: `python path/to/folder/main.py`.

## Functionalities

1. **User Registration**: Create a user profile with personal details.
2. **Habit Creation**: Define multiple habits (with a task specification and periodicity).
3. **Task Completion**: Check-off tasks at any point in time.
4. **Analytics**: Analyze habits, longest streaks, and more.
